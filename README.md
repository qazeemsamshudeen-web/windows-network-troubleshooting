# Windows Network Troubleshooting Lab

A beginner hands-on project showing Windows network troubleshooting using built-in commands.

## Objective

Understand how Windows handles IP addressing and basic troubleshooting.

## Tools Used

- Windows Command Prompt
- ipconfig
- Cisco Packet Tracer

## Commands Used

- #ipconfig /all
- shows current ip configuration
- ![ipconfig all](images/ipconfig.all.png)
- used ipconfig /all  to inspect the system network configuration, including IPV4 , subnet mask, DHCP Details, Default gatway
- #ipconfig /release
- release current ip address and DHCP lease
- ![ipconfig.all](images/ip.release.png)
- ipconfig /renew
- refresh DHCP server(router) to get a new ip address
- ![ipconfig.all](images/ipconfig.all.png)

## Learning Outcome

- Learned how to inspect IP configuration
- Learned DHCP renewal process
- Practiced troubleshooting workflow
- Documented screenshots and results
